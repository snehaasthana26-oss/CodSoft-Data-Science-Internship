# CodSoft-Data-Science-Internship
Data Science projects completed during CodSoft internship
## ✅ Task 1 — Titanic Survival Prediction

**Dataset:** Titanic passenger data  
**Goal:** Predict whether a passenger survived  
**Model:** Logistic Regression  
**Accuracy:** 81%  

**What I did:**
- Explored and cleaned the dataset
- Handled missing values in Age and Embarked
- Encoded categorical features (Sex, Embarked)
- Split data into train/test sets (80/20)
- Built Logistic Regression model
- - Evaluated using accuracy, confusion matrix

**Key Finding:** Sex and Pclass were the 
most important survival predictors
